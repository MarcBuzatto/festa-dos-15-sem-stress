# Pasta de Produtos

Coloque os arquivos do kit nesta pasta com EXATAMENTE estes nomes:

| Arquivo | Produto |
|---|---|
| `guia-completo.pdf` | Guia Completo — 12 Capítulos |
| `checklist-fases.pdf` | Checklist por Fases — 12 Meses |
| `planilha-orcamento.xlsx` | Planilha de Orçamento |
| `bonus-cronograma.pdf` | Bônus: Cronograma do Dia da Festa |
| `bonus-perguntas.pdf` | Bônus: Perguntas para Fornecedores |

Após colocar os arquivos, faça `git add . && git commit && git push` para publicar na Vercel.
